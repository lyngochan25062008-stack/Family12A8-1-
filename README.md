# Global Family Website

## Mô tả
Trang web kết nối các gia đình trên toàn thế giới, chia sẻ văn hóa và câu chuyện gia đình.

## Tính năng chính
1. Bản đồ tương tác hiển thị các gia đình từ nhiều quốc gia
2. Form chia sẻ câu chuyện gia đình
3. Hiển thị gia đình tiêu biểu
4. Lịch sự kiện cộng đồng
5. Responsive design cho mọi thiết bị

## Công nghệ sử dụng
- HTML5, CSS3, JavaScript (ES6)
- Font Awesome cho icons
- Google Fonts (Inter, Playfair Display)
- SVG cho bản đồ tương tác

## Cài đặt & Chạy
1. Tải toàn bộ thư mục `global-family`
2. Mở file `index.html` trong trình duyệt
3. Không cần cài đặt server hay cấu hình phức tạp

## Tùy chỉnh
1. **Thay đổi màu sắc**: Chỉnh sửa biến CSS trong `:root` ở file `style.css`
2. **Thêm nội dung**: Cập nhật HTML trong các section tương ứng
3. **Thêm hình ảnh**: Đặt hình ảnh vào thư mục `images/` và cập nhật đường dẫn
4. **Thêm ngôn ngữ**: Mở rộng đối tượng `translations` trong `script.js`

## Các bước phát triển tiếp theo
1. Tích hợp backend với Node.js/Python để lưu dữ liệu
2. Thay thế bản đồ SVG bằng Leaflet/Google Maps
3. Thêm tính năng đăng nhập/tài khoản
4. Tích hợp diễn đàn thảo luận
5. Thêm hệ thống tin nhắn giữa các gia đình
6. Tích thanh toán cho các sự kiện có phí
7. Tối ưu SEO và hiệu suất
